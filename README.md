# HTMLInpuTtypes

A tiny packacge to specify HTML input types in a cleaner way.

# Usage

# Install package
npm install -save html-inputtypes

# import package
import INPUT_TYPES from 'html-inputtypes';

# use package (React example)
<input type={INPUT_TYPES.TEXT} />
